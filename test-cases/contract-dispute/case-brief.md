# 案件概要

## 基本信息
- **案由**: 民间借贷纠纷
- **原告**: 张某（自然人）
- **被告**: 李某（自然人）
- **标的额**: 人民币 500,000 元

## 案件简述
2023年6月15日，被告李某因经营需要向原告张某借款50万元。双方签订了《借款合同》，约定借款期限12个月（至2024年6月15日届满），年利率12%，按月付息，到期还本。

原告通过银行转账方式向被告交付了50万元借款。被告在前3个月（2023年7-9月）正常支付利息共15,000元，此后未再支付利息，亦未在到期日归还本金。

原告多次催收未果，遂向法院提起诉讼，请求被告偿还借款本金50万元及逾期利息。

## 争议预判
1. 借款合同的效力
2. 利率约定是否合法
3. 已还利息的认定
4. 逾期利息的计算标准
